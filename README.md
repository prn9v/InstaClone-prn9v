🚀 InstaClone

A sleek and modern full-stack Instagram clone built with the powerful MERN stack (MongoDB, Express.js, React, and Node.js). This project brings you the essential features of a social media platform, including authentication, post creation, and user interaction.

🌟 Features

✅ User Authentication (Signup/Login)
✅ Post Creation (Upload Images)
✅ Like & Comment on Posts
✅ Follow/Unfollow Users
✅ User Profile Pages
✅ Responsive UI

🛠 Tech Stack

🎨 Frontend:

⚛️ React.js (with Hooks & Context API)

🎨 Tailwind CSS for styling

🔄 React Router for navigation

🖥 Backend:

🏗 Node.js with Express.js

🗄 MongoDB with Mongoose

🔐 JWT Authentication

📂 Multer for Image Uploads

☁ Cloudinary (Optional) for storing images
